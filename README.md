## Usage

1. Adjust the password length using the slider (6-30 characters)
2. Toggle "Numbers" checkbox to include numeric characters
3. Toggle "Character" checkbox to include special characters
4. The password will automatically generate based on your selections
5. Click the "Copy" button to copy the password to your clipboard
